# 神奈川 Discovery 引き継ぎメモ（別Claudeセッションより, 2026-07-08）

## これは何
神奈川の残り市町村について、別セッション（監修営業側）で Discovery（各自治体の
市/町/村指定文化財の公式一覧URL特定）を行い、その成果を `kanagawa_c.jsonl` に置いた
ものです。**メイン（文化財を回している）側への引き継ぎ用**。build/fetch/extract は
競合破損（build_data多重起動）を避けるため、こちら側では一切実行していません。

## 収録状況（引き継ぎ時点）
- `kanagawa_a.jsonl` + `kanagawa_b.jsonl` = メイン側が収集中（引き継ぎ時点で13自治体、随時拡大中）
- `kanagawa_c.jsonl` = 本引き継ぎ分 **20自治体**（a/bと重複しないものだけに絞済み）
  - 三浦・伊勢原・南足柄・厚木・大和・小田原・山北・座間・愛川・海老名・清川・湯河原・
    真鶴・秦野・箱根・綾瀬・茅ヶ崎・藤沢・逗子・開成
- a/b(13) + c(20) = 神奈川 全33自治体 をカバー

## 取り込み前に必ず（重要）
- **メイン側は a/b を高速で拡大中**。短時間で平塚・鎌倉・松田等を a/b 側に吸収した実績あり。
  fetch前に **municipality（自治体名）で a+b+c を重複除去**すること。同一自治体がa/bとcに
  二重で入ると二重fetch/二重抽出の恐れ。
- 既に a/b が33自治体に到達しているなら、`kanagawa_c.jsonl` は破棄してよい。

## 特記事項（strategy/件数の根拠）
- 松田町: 公式サイト・例規集ともに町指定一覧なし（冊子のみ）→ 本引き継ぎでは除外済み（a/b側でnone記録済）。
- 真鶴町: 公式HTML一覧なし。教育委員会資料『まなづるの統計』PDFの「町内の国・県・町指定文化財」表を採用（strategy=pdf, 町指定112件）。
- 湯河原町: 統合HTML一覧なし。公式冊子PDF『湯河原町の文化財』を採用（strategy=pdf）＋史跡カテゴリHTML補助。
- 鎌倉市: カテゴリ別PDF提供の目次型（strategy=pdf 相当。a/b側で収集済につきcからは除外）。
- official_count はページ/表に明記の数値のみ。非明記は null。

## 出所
Discoveryは公式ソース（city.*.kanagawa.jp / town.*.kanagawa.jp / *.lg.jp / 教育委員会 / 公式統計PDF）のみ。
GET/閲覧のみ・受動的に確認。各URLは実在確認済み。

# 福岡県 市指定文化財 ギャップフィル記録（2026-07-12実施）

対象：福岡市（fukuoka）、久留米市（kurume）、宗像市（munakata）

## 福岡市 (fukuoka): 0 → 228行

- sources/fukuoka.jsonl記載の`bunkazai.city.fukuoka.lg.jp/sp/cultural_properties/search/`（JS/AJAX検索）をまず静的解析。robots.txtは`/qr.php`等の一部パスのみDisallowで、`/cultural_properties/`配下は許可されている。
- PCサイト版（`/cultural_properties/search/?from_sp=1&page=N`）のHTMLソースに`var datas = [...]`というJS変数として、各ページ10件分のレコード全体（名称・区分・指定区分master_shitei・種別master_kubun/ku_type・所在地・指定年月日・cid）がJSON形式で埋め込まれていることを発見。ページネーションはGETクエリのみで完結。
- 全35ページ（2.6秒間隔）をpassive GETで取得し、重複排除の上349件のユニークレコードを収集（`page=36`以降は404で終端確認）。うち`master_shitei.title == "市指定"`は83件のみだった。
- **重要な発見**: このオンライン検索DB（349件）は福岡市文化財活用課が運営する検索システムだが、市指定文化財の網羅性は低い（83/約228＝36%）。福岡市文化財ページの別の解説記事は市指定429件と述べているが、これは登録文化財等を含む可能性のある古い数値と判断（後述の一次資料と整合しない）。
- そこで福岡市教育委員会発行の**令和7年度教育要覧・資料編**（`https://www.city.fukuoka.lg.jp/kyoiku-iinkai/k-seisaku/ed/documents/09_R7siryou.pdf`、全42頁PDF）を発見。p.90〜106（PDF内通し22〜38頁目）に「福岡市内指定・登録文化財一覧」という一次資料の詳細表（区分＝国/県/市、名称、員数、指定年月日を全13カテゴリ：建造物・絵画・彫刻・工芸品・書跡典籍古文書・考古資料・歴史資料・無形文化財・有形民俗文化財・無形民俗文化財・史跡・名勝・天然記念物）を掲載。登録文化財（(14)節）は指定ではないため除外。
- pdftotext -layoutで抽出したテキストを正規表現でパース（`区分 名称 員数 指定年月日`のカラムを2スペース区切りで分離）し、「市」区分の行のみ抽出→**228件**。タスク指示の期待値（市指定約227件）とほぼ一致し、こちらを一次資料・主データソースとして採用。
- オンライン検索DB（83件）とPDF一次資料（228件）を名称の正規化（括弧内注記除去）で突合し、67件で所在地（location）情報を補完。残り161件はPDFに所在地欄がなくlocation=null。
- 和暦→西暦変換は明治=1867+、大正=1911+、昭和=1925+、平成=1988+、令和=2018+のオフセットで実施、全件が2026-07-12以前であることを確認。
- description欄は一次資料に短評（概要文）がなく、null-over-guessに従いnullで統一。
- ソース：`https://www.city.fukuoka.lg.jp/kyoiku-iinkai/k-seisaku/ed/documents/09_R7siryou.pdf`（PDF、pp.90-106）、補助的に`https://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/{id}`（HTML、location補完のみ）

- CSVの「文化財分類」列で「市有形文化財」「市有形民俗文化財」「市無形民俗文化財」「市天然記念物」「市史跡」「市名勝」に絞ると**107件**、公式集計（107件）と完全一致。

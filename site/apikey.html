Get a free API key / APIキーの発行

Instant and free — just an email address. No verification email is sent; signing up again with the same email returns the same key.
メールアドレスを入力するとその場でキーが発行されます（無料・確認メールなし）。 同じメールアドレスで再発行すると常に同じキーが返るため、紛失時は再入力してください。

Your API key / あなたのAPIキー

Usage / 使い方


        

          Send the key as an x-api-key header (recommended) or a ?key=
          query parameter. Only /api/v1/* requires a key — bulk files under
          /data/* and this map site stay keyless.

          キーは x-api-key ヘッダ（推奨）または ?key= クエリで送ります。
          キーが必要なのは /api/v1/* のみで、/data/* のバルクデータと
          地図サイトはキー不要です。

          Note: the key encodes your email address (base64url) — do not post it publicly.

          注意: キーにはメールアドレスが埋め込まれています。公開の場に貼らないでください。
